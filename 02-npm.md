Node Package Manager (npm) - the standard package manager for Node.js
>> Yarn and pnpm are alternatives to npm cli

# Init
npm init

# Install Package
npm install
npm install <package-name>
npm install --save-dev <package-name>

>> install flags
--save (-S)
--save-dev (-D)
--save-optional (-O)

# Update package
npm update
npm install <package-name>@<version>

# Running task
npm run <task-name>
